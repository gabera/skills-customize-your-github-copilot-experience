# 📘 Assignment: Hangman Game

## 🎯 Objective

Build a classic word-guessing game where players reveal hidden words by guessing letters before their attempts run out. You will practice string manipulation, loops, conditionals, and random selection in Python.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Create a Hangman game where players guess letters one at a time to reveal a randomly selected word before exhausting their maximum attempts.

#### Requirements
Completed program should:

- Select words randomly from a predefined list
- Accept letter guesses and display current progress in the format: `_ _ _`
- Track remaining incorrect attempts
- End when the word is guessed or attempts are exhausted
- Display victory or defeat messages
